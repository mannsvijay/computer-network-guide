⚠️ Tricky Topics Students Usually Get Wrong
CIDR Address Aggregation
120, 37, 49, 40 hosts problem (2024 Main C1) — log commonly galat address size calculate karte hain. Next power of 2 approach use karo.
ALOHA Throughput Formula
Pure ALOHA = 18.4%, Slotted = 36.8% — log formula yaad karte hain but derivation galat karte hain. S = Ge^(-G) formula samjho.
Dijkstra vs Bellman-Ford
Dono routing mein confuse karte hain. Dijkstra = shortest path (Link state), Bellman-Ford = distance vector. Table banake practice karo.
TCP 3-Way Handshake vs 4-Way Teardown
Connection establish (SYN, SYN-ACK, ACK) aur termination (FIN, ACK, FIN, ACK) mix ho jaata hai. Diagram se yaad karo.
M/M/1 vs M/M/m Queuing
M/M/1 mein single server, M/M/m mein m servers. λ/μ = ρ (traffic intensity). Log formulas mix karte hain — U2 properly padho.

Unit 2 — Queuing Theory Formulas
Little's Formula
L = λW   |   Lq = λWq   |   W = Wq + (1/μ)
Traffic Intensity
ρ = λ/μ   (must be < 1 for stable system)
M/M/1 Queue
P(0) = 1 - ρ   |   P(n) = (1-ρ)·ρⁿ
L = ρ/(1-ρ)   |   Lq = ρ²/(1-ρ)
W = 1/(μ-λ)   |   Wq = ρ/(μ-λ)
M/M/1/K Queue (Finite Buffer)
P(0) = (1-ρ)/(1-ρ^(K+1))   |   λ_eff = λ(1 - P(K))
